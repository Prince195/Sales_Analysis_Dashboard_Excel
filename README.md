# 📊 Excel Dashboard Project – Sales Analysis

## 📌 Project Title

Sales Data Analysis and Interactive Dashboard using Microsoft Excel

---

## 📁 Dataset Description

This project uses a structured sales dataset containing the following fields:

* Customer_ID
* Customer_Name
* Region
* Product_Category
* Sales
* Quantity
* Total Sales
* Discount (%)
* Order_Date
* Profit
* Total Purchase
* Total Cost

The dataset is used to analyze sales performance, profit trends, and customer behavior.

---

## 🎯 Objective

The main objectives of this project are:

* To clean and prepare raw data
* To perform data analysis using Excel formulas
* To create Pivot Tables for summarization
* To build an interactive dashboard
* To derive business insights from sales data

---

## 🧹 Data Cleaning & Preparation

* Removed duplicate records using Excel tools
* Handled missing values using IF conditions
* Converted Discount (%) into numeric format
* Ensured correct data types (Date, Number, Text)

---

## 📊 Data Analysis Techniques

### 1. Basic Calculations

* Total Sales = Sales × Quantity
* Discount Amount = Total Sales × Discount
* Net Revenue = Total Sales – Discount

### 2. Conditional Analysis

Used IF functions to classify:

* Profit Status (High / Low)
* Discount Category (Low / High)
* Sales Performance (Excellent / Good / Average)

---

## 🔍 Advanced Excel Functions

### ✔ VLOOKUP

Used to fetch target values based on product category

### ✔ INDEX-MATCH

Used as an alternative to VLOOKUP for better flexibility

### ✔ Nested IF

Used for multi-level decision making

---

## 📈 Pivot Table Analysis

Created Pivot Tables to analyze:

* Total Sales by Region
* Profit by Product Category
* Quantity sold per Customer
